Git

Git yra populiariausia kodo versijavimo programa.

Git komandos

  • git init - sukuria git repozitorija dabartiniame aplankale
  • git status - nurodo informacija apie lokalios Git'o repozitorijos statusa
  • add - prideda nurodytus failus prie commit paruotu failu(stage'ina)
  • rm --cached - isima nurodytus failus is stage'inimo vietos/zonos (unstage'ina)
  • commit -m (minus m) "zinute" - sukuria kodo versija is stage'intu failu (o tas minus m) prideda zinute apie pakitimus (zinute trumpa ne ilgesne nei 50simboliu ir aiski)
  • restore - grazina nurodytus failus i buvusio commit'o busena
